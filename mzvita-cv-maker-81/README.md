# MozVita CV Builder

MozVita is a modern CV builder designed for the Mozambican market. It features a robust multi-page editor with real-time preview and export capabilities.

## Technologies Used

- **Framework**: React / Vite
- **TypeScript**: Typed language for reliability
- **Styling**: Tailwind CSS & Vanilla CSS
- **Components**: shadcn/ui (Radix UI)
- **Database/Backend**: Supabase

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <YOUR_GIT_URL>
   cd mzvita-cv-maker-81
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

## Key Features

- **Multi-page Editor**: A4-sized "sheet stack" architecture allowing for real paper-like feel.
- **Templates**: Various professional templates specifically optimized for different user segments.
- **Mobile Responsive**: Fully usable editor and navigation for smaller devices.

## Supabase Configuration

This project uses Supabase as its primary backend. All resources and assets are managed securely through direct integration with Supabase Services and local Git management for static assets.
